the top 40 charts

providing the top 40 UK chart data in json, xml and serialized

i've been providing the top 40 charts in various formats for a long time now. it began as a bbc backstage project but i've carried it on over the years. feel free to use this as you wish. the chart data is obtained via the bbc's site and transformed into various machine readble formats.

In JSON format:

http://{{ vars.server.server_name }}/top40.json
In XML format:
http://{{ vars.server.server_name }}/top40.xml
In serialized form:
http://{{ vars.server.server_name }}/top40.ser

let me know if you need any more or want something else done. as usual all this stuff is available at source from my framework- @ouchy
