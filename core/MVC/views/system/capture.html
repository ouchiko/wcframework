screen shot generation api

capture any website you want in one simple api call to the webcoding screen capture service. this service has very little requirement for extensive documentation. call the service with a uri, a width and the return format and the system will send back an image.

http://{{ vars.server.server_name }}/capture/request/?uri={{ "http://www.bbc.co.uk"|url_encode }}&width=500&format=jpg

uri should be a valid, url encoded URI. width should be the resulting size image you want in pixels. Use * for a full size render. lastly format determines the response image format and can be either png or jpg
