News over time

for a period of time i captured and stored a vast selection of the BBC news website

over a number of years and through various iterations of c# application through to linux screen capture i managed to capture a significant amount of the bbc's homepages and news sites as part of thier original project bbc backstage. you can see the old calendar site here as an example. even older, if you fancy a trip down some old pages is the original bbc archive i did here.

after collecting the data i collated it into an interesting summary video, in this case it was targeted to show the bbc over a random period and then a seperate video on the london riots back in 2011.

the result of this effort was the creation of the screen capture tool found here which has an API where you can request any uri with specified dimensions and it will go away and grab that for you.
