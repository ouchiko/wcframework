Docker containers

experimenting with and making use of the excellent docker

docker is awesome. imagine building your own distributed system which works for your code. you can run it anywhere you want.

this site is run on two docker containers via digitalocean. the development of this website was done on an ubuntu installation running docker via vagrant. how cool is that, developing on your local computer and distributing your code and containers together ensuring everything just works.. evert time. no more developing on some local configuration that you have trouble getting into a live environment.

truthfully, i'm in my infancy with this technology. i'm learning about it every day and trying to understand more about the possibilities and potential downsides to allowing developers to become more devops.

feel free to check out my current docker implementation via github
